# licenses/

Файл `LICENSE` в корне репозитория содержит лицензию проекта Framehold Engine: GNU Affero General Public License version 3 or later.

`THIRD_PARTY_NOTICES.md` содержит attribution и inventory для намеренно включенных или адаптированных third-party components and assets.

Каталог `licenses/` может позже содержать копии required third-party license texts, если dependency или asset требует bundled license text или notice preservation.

Third-party components остаются под собственными лицензиями. Добавление dependency, vendored file, font, icon set, visual asset или adapted material может потребовать добавления его license text и обновления `THIRD_PARTY_NOTICES.md`.

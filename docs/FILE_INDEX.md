# FILE_INDEX

## Корень репозитория

- `README.md` — краткое описание проекта, его статуса и ссылки на документацию.
- `AGENTS.md` — правила для будущих агентских и Codex-сессий в этом репозитории.
- `.gitignore` — правила исключения локальных артефактов, секретов, медиа, кэшей и IDE-local state из Git.

- `docs/DATA_MODEL.md` — черновая модель данных и открытые решения.
- `docs/PERMISSIONS.md` — роли и правила доступа.
- `docs/PUBLIC_SITE.md` — структура публичного сайта и URL.
- `docs/ADMIN_WORKFLOW.md` — административные процессы и сценарии работы.
- `docs/UI_DESIGN.md` — визуальное направление публичного frontend.
- `docs/TECH_STACK.md` — выбранный стек и невыбранные альтернативы.
